# hylerrix: 我的个人简历

* 网页版简历：开发中

***使用方式***

- 0、`yarn add` 或 `cnpm install` 安装依赖
- 1、编辑 `/src/index.html` 文件
- 2、命令行运行 `yarn run rebuild`
- 3、打开 `/index.html` 进行预览

> chrome 浏览器 - "打印" - 设置无边距 - 即可导出
