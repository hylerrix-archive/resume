* Fork或借鉴请注明出处 [@ Cong Min](https://congm.in)

#### Usage
- 0、`yarn` 安装依赖
- 1、编辑 `/src/index.html` 文件
- 2、命令行运行 `yarn run rebuild`
- 3、打开 `/index.html` 进行预览

> chrome浏览器 - "打印" - 设置无边距 - 即可导出
